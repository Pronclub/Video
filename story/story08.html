මගේ නම අකිල මේ සිද්දිය උනෙ මන් කෝස් එකක් කරන්න කොලබ ගියාට පස්සේ මුලිනම මයි අප්පචි කැම්පස් එකට ගිහින් නව්තින්න බෝඩිමක් හෙව්වා එත් හම්බුනෑ හැම එකක්ම වගෙ දීලා තිබුනේ වයසක සීයා කෙනෙක් හම්බුනා අපිට එ සීයා ට කිවම අපිව එක්කන් ගියා ගෙදරකට අම්මෝ මෙන්න බෝලේ සුදුම සුදු අක්ක කෙනෙල් ලමයෙකුත් වඩාගෙන එලියට ආවා

සියා විස්තරේ කිවම එ අක්ක අපෙ නොම්බර් එක ඉල්ලගත්තා අයියගෙන් අහලා කියන්නම් කියලා ඉට පස්සේ අපි කොලබ ලොකු අම්මලගෙ ගෙදර ගියා අප්පචි ගෙදර ගියා රෑ මන් බොඩිමක් ගැන කල්පනාලා කර කර ඉද්දි කෝල් එකක් අවා අදුරනැති නොම්බර් එකකින් අර අක්කල්ගෙ ගෙදරින් හැබැයි අයියා කතා කරේ උදෙට එන්න කිවා නිතී ගොඩක් දැම්මා රෑ වෙලා එන්න බෑ කිවා යාලුවෝ එක්කන් එන්නත් බෑ කිවා වෙන කරන්න දෙයක් නැති නිසා මන් කැමති උනා.

දවස් දෙක තුනක් ඔම ගෙවිලා ගියා කොම හරී පලවෙනි සතිය ගෙවිලා ගියා ඉරිදා නිවඩුනේ මන් බොඩිමට වෙලා කම්මැලි කමේ හිටියේ ආ කියන බැරි උන මන් උඩ තට්ටුවෙ හිටියෙ කමරෙක අක්කලා යට තට්ටුවේ මන් කමැලි කම්ට ඉතිම් ජනෙලෙ ඇරියා දෙයියනේ මන් දැකපු දේ ගෙදර අය නාන තැන තියෙන්නෙ මන් ඉන්න කමරෙට පෙන්වා අක්ක රෙදි සොදන්වද කොහෙදෝ මන් අයම ජනෙලෙ වැහුවා හරිනැති නිසා එත් කම්මැලි නිසා අය පොඩ්ඩක් ඇරලා බල්න් හිටියා අම්මො එ පර අක්ක දිය රෙද්ද ඇදලා නාන්න හදන්වා මන් හොරෙන් බලන් ඉන්නවා අක්ක දන්නෑ සුදු පාට පිට හොදට පෙනව අක්කා කොන්න්ඩෙ ට ශැම්පෝ දාලා අතුලන්වා මගේ මල්ලියත් ටික ටික කෙලින් වෙගන් එන්වා ඔය අතරේ

මන් බස් එකෙ ඉද්දී අදුරනැති නොම්වරෙකින් කොල් එකක් අවා මන් ආන්සර් කරා අම්මො අක්කගෙ කටහඩ ඔයා එන්නැද්ද් මල්ලි රෑ වෙලා ගෙට් එක වහන්න ඔනි කිවා අයියා බනින්වා කිවා මන් එනගම්න් ඉන්නෙ කිවම කට් කරා එදා තම මට අක්කගෙ නොමබර් එක හම්බුනේ ඔන්න දවසක් රෑ මන් කම්මැලි කමෙ ඉද්දී මට හිතුන අක්කට මැසෙජ් එකක් දාන්න එත් බයයී හිත් හිත ඉද්දි ඔන් දොරට තට්ටු කරන සද්දයක් අවා මෙන්න බොලෙ අක්කා අයිස් ක්‍රිරිම් එකක් ගෙනත් මට දිලා කන්න කිය්ල ගියා

මන් කල ඉවර වෙලා මැසෙජ් එකක් දැම්ම ඉමෝ වලින් අක්ක අයිස් එක රහයි කොප්පෙ ගෙනත් දෙන්නද කියලා එයා කිවා උදෙට ගමු කිය්ලා ඔන්න දැන් අක්ක උදෙට gm msg එම දනවා කැම එම විශෙශ දෙයක් හැදුවොත් මට එවන්වා උඩක කියන්න බැරි උනනෙ අක්කට බබෙකුත් හිටියා පොඩී දවසක් රෑ අයියා නැති දවසක මන් අක්ක් එකක මැසෙජ් කරා අක්කා : නෑ මල්ලි බබා නිදි කරනවා මම : අයියෝ පුතා වද දෙන්වා නෙද අක්කා : ඔව් අනේ කිරි බොන්වා ඉවරයක් නෑ අක්කා : කිරී කොහෙද අනේ තියෙන්නෙ දැන් මෙයා ලොකු ලමයෙල්නේ බොරුවට බිබි ඉන්නව මම : අයියො සුදුපටට තියන එකෙ බොන්න ආස ඇති එ ලම්මයා (හුටා මෙ මොන්වද මන් කියන්න්නෙ) අක්කා : කොහෙද සුදු ඔයා දැක්කද් ආආ අක්කා : අනෙ යනවා පුතට මන්.දවල්ට කිරි දෙන්නැ ඔයා කොම දකින්න්ද මොඩයෝ අක්කා : අනෙ යන්න බොරු නොකියා මම : බොරු නෙවෙ බනිනැත්ම් කියන්නම් මම : ඔයා නානවා උඩට පෙන්වා අනෙ මම : බොරු නෙවෙ ඔයගෙ දිය රෙද්ද නිල් පටයිනෙ අක්කා : අනෙ දෙයියනේ වල් කොල්ලා නනව බලන්වද මම අය ගෙ ඇතුලෙන් නානවා මම : පිස්සුද අනෙ පෙන්නැ ම්න් බොරුවට කිවෙ අක්ක : එනම් නිල් පාටයි කිවේ මම : පිට විතරයි අනෙ පෙන්නෙ අක්කා : එනම් සුදු පටයි කිවෙ මොඩයා මම : පිට සුදුයිනෙ කුක්කූ දෙක ඔයිට වඩා සුදු ඇති හික්ස් අක්කා : වල් කොල්ලා ආය එම බලන්නැ අක්කා : මන් නිදා ගන්න්වා gn මම : සොරි අක්කෙ විහිලුවක් කරෙ අක්කා : හරි මොඩයෝ නිදිමතයි එකයි මම : එනම් gn අක්කා කිවා රිප්ලයි අවෙ නෑ

උදෙ මන් ලැජ්ජවෙන් මුන බැලුවෙ අක්කගෙන් අක්ක හින්වෙලා යද්දි බයි කිවා ඔන්න් ඉගව ඉරිදා උදා උනා මන් ජනෙලෙ ඇරියා අක්ක නන්න එනක්ම් බලගෙන හිටියා අද අයියත් ගෙදර අක්ක නනන අවා ඔන් මන් අක්කට බලගෙන ඉන්වා කිය්ල අගවන ජනෙලෙ හයියෙන් ඇරියා අක්ක දැක්කෙ නැ වගෙ නනවා මෙන්න බොලෙ වෙන්දට නිල් චිත්තෙ ඇද්න් නන ගැනි යට සායක් ඇදම් නානවා මට හිතුනා අක්ක මට පෙන්න ඇන්ද කියලත් අම්මො එත් ඉතිම් මට පෙන්නෙ පිට විතරනේ.

අවාසනවාක මහත අක්ක සබන් ගගා අතුලනවා කුක්කු මර අත් වලින් පෙන්වා එත් කුක්කු පෙන්නෑ මන් අතෙ ගහන්වා සුපුරුදු විදියට නාල ඉවරයි අයියගෙ කාර් එකෙ සද්දෙ ඇහුනා අයියා බැග් එකකුත් අර්ගෙන කොහෙ හරි ඇතක යන්න වගේ ගියා ඔන් මන් අක්ක්ට මැසෙජ් එකක් දැම්මා

අක්කා : ඔව් මල්ලී වැඩ වගෙක්ට කුරුනැගල ගියා අය එන්නෙ සතියකින් කිවා (මගෙ ඉහෙ මලක් පිපුනා වගෙ) මම : අයියෝ අක්ක්ට තනියම ඉන්න වෙන්නෙ අක්ක : ඇයි ඔයා ඉන්නෙ.මොඩයෝ.උඩ අක්කා : ඔව් ඉතිම් චන්න්ඩියා මැසෙජ් කරට මුන දැක්කම ලැජ්ජවනෙ අක්ක : ඔව් ඔව් මන් දන්නැද්ද අද රැට කඩෙන් කන්න එපා මන් ඔයටත් එක්ක උයනව මම : අම්මො අද රසට කන්න පුලුවන් දැන් අක්ක මත් එක්ක ෆිට් රැ 8ට විතර පොඩී එක අතෙ බත් එකක් උඩට එව්වා මන් කල10ට විතර මැසෙජ් එකක් දැම්ම මම : කව්ද දන්නැ ලැජ්ජ් බත් එක පුතා අතෙ එව්වෙ අක්කා : අනෙ මන් කල ටිකක් ඇග සොදගත්තා මල්ලි ඔයට වඩගිනි ඇති කියලා එව්වෙ බබා අතේ අක්කා : කිරි බොන්වා අනෙ නින්දා යගෙන් එන්නෙ එයට මම : අපිට කොයින්ද ඉති කිරි දීලා නිදි කරන්න කට්ටිය අක්කා : ඇයි ඔයා ලොකුයිනේ මොඩයා මම :අපිට කොයින්ද ඉති කිරි දීලා නිදි කරන්න කට්ටිය අක්කා : ඇයි ඔයා ලොකුයිනේ මොඩයෝ අම්මාට දැන් බැනෙ ඔයා ගව ඉන්න මම : හ්මම්ම්ම්ම්ම්ම්ම්ම්ම්ම් අක්කා : සොරී මන් නිකම් කිවේ අක්කා : පිට විතරද දන්නෑ මොඩයා මම : අනිත් එවා පෙන්නැනෙ ඉතිම් අක්කා : හිනෙන් තාමා මොඩයෝ මල්ලි නිදී මම : ආ එනම් ලැජ්ජ නැත්තම් උඩට එන්න අක්කා : උඩ කොල්ලෙක් ඉද්දි කොම එන්න්ද මොඩයොරැ වෙලා අක්කා : රැට අදින එක ඇද්ම් ඉන්නෙ දැන් එන්න බැ අක්කා : ඉන්වකෝ මොඩයා එක්ක බැනෙ

සීන් උනා රිප්ලයි නෑ ටිකකින් දොරට තට්ටුකරන්වා දෙයියනෙ අදන්ම් අක්ක්ට යන්න දෙනෑ කියලා හිතගෙන මන් එන්න ඇතුලට කියලා ඇදට වෙලා හිටියා එයා දොර ඇරලා එක පරටමවැහුවා මන් ඇයි ඇහුවා ටී එකක් දාගන්න්වා මොඩයෝ කිවා මම : එන්න අනෙ ඇතුලට ටී ශර්ට් නෑ අක්කා : මේ කොල්ලන්ම් ඉන්න විදිය මම : මෙ කලිසමක් ඇද්ම් ඉන්නෙ පෙන්නැද්ද් අක්කා : මොඩයා කිය කිය හිනා වෙවී කිවා මන් අවනෙ දැන් යන්නම් කිවා මම : අනෙ මෙ එන්න ඇතුලට දොර ගවා මුර කරන්වද ලැජ්ජ කරී අක්ක : මට ලැජ්ජ නැ අනෙ කියල ඇතුලට අවා මන් කිවා ඇදෙ වාඩිවෙන්න කියලා අක්කා ඇද්ලහිටියේ පටි වලින් බදින නයිටියක් හැබැයි මොකුත් පෙන්නෑ මගෙ හොදටම නැගලා මන් තාම ඇදෙ නිදාගෙන ඉන්නෙ මන් හිමිට කොට්ටය්ක් තියගෙන ඉදගත්තා අක්කා ගවින් මම : අයියා එන්නැද්ද් සුවර්ද මට දැන් පෙරල්ගෙන ගහන්න හිතෙන්වා ඉවසන් හිටියා මන් මම : ඓනවා කියගෙන ලගටගියාඅතට මුන ලන් කරල සුවදයි කිවා මම අක්කගෙ නයිටියෙ පටියෙන් ඇද්දා එක ලිහුනා මම : කුක්කු බොන්න හදන්නෙ කිවා මන් දැන් ගත්තා දැන් හරී කිය්ලා මන් හිම්ට කුක්කු එකක් ඇල්ලුවා අක්කා ඇස් පියාගත්තා මන් අක්කව ඇදෙ පෙරල ගත්තා අක්ක කෙදිරි ගනාව මන් හිම්ට තොල් කිස් කර අක්කත් පටන් ගත්තා විනඩි දහය්ක් විටර කිස් කරා ඉට පස්සෙ මන් අක්ක්ගෙ නයිය ගැලව්වා.

අම්මෝ සුදු කුක්කු මන් මෙච්චර කල් නොදකපු එවා මන් මිරික මිරික බොන්න ගත්තා අක්ක පිස්සෙක් වගෙ කෙදිරි ගනවා මන් කිස් කර කරම පහාලට ආවා ඇවිත් හුත්ත්ට ඇගිල්ල දාම්ම් අක්ක් කෑ ගහ්න්වා මන් මගෙ කෑලට විතරයි දිව දන්නෙ මට අනිත් එවා අපිරියයි ඇත්තටම මන් දිව දාන්න ගියෙ නැද් ඇගිල්ලෙන් කරා ඉට පස්සෙ අක්කා මවා පෙරලගෙන මගෙ මල්ලිව අතුන් හුරත්ල් කරන්න ගත්තා කටෙ දාලා සූපු කරන්න ගත්තා මට යන්න වගෙ මන් ඉට පස්සෙ අක්ක්ව අය පෙරලගෙන දාන්නද කිය්ල ඇහුවා හිම්ට දාන්න රත්තරම් මට ඉන්න බැ කිවා මන් කිවා කොන්ඩම් නැනෙ කියලා මෙන්න මෙකි කියන්ව ලුපකයක් දාලා තියෙන්නෙ බය වෙන්න එපා දාන්න කිවා
